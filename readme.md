# 🌐 YUVATA — Youth Digital Literacy Assessment Platform

> *"Growing up with technology ≠ digital literacy."*

**YUVATA** is an AI-powered digital literacy assessment and mentoring platform designed for youth aged 13–25. It evaluates real-world digital skills — not just tech familiarity — and provides personalized learning roadmaps to bridge skill gaps.

Built for the hackathon problem statement: *Does growing up with technology automatically lead to digital literacy?*

---

## 🧠 Design Thinking Approach

**1. Empathize: Who suffers?**
Our target users, youth aged 13–25 (often assumed to be "digital natives"), suffer. While they have widespread access to technology, they frequently lack true digital competence. This leaves them extremely vulnerable to phishing, misinformation, deepfakes, privacy breaches, and online scams without them even realizing it. 

**2. Define: What exact problem?**
There is a massive gap between *technology access* and *technology competence*. Existing digital literacy tools are generic, non-adaptive, test outdated skills (like what a mouse is), rather than real-world internet safety, and fail to engage the modern youth demographic effectively. Technology moves faster than current education.

**3. Ideate: What are possible solutions?**
A free, youth-centric, AI-adaptive platform that doesn't just evaluate but mentors. Users should be able to self-select relevant skill areas (like Social Media Literacy or Deepfake Spotting) and take a fast, scenario-based test that identifies their specific blindspots and offers a personalized roadmap.

**4. Prototype: Build minimum viable solution.**
We built **YUVATA** — an interactive web app featuring a premium, gamified UI. It features a skill selection onboarding screen, dynamic scenario-based questions generated by AI (so they don't get stale), an intuitive results dashboard with a radar chart, and a conversational AI mentor to explain incorrect answers and guide the user dynamically.

**5. Validate: Does it actually work?**
We measure success by tracking assessment completion rates (target: < 10 mins completion time), user engagement with the AI coach, and the accuracy of the AI's skill gap analysis compared to manual evaluation. The scalable platform allows us to validate if users are learning from their mistakes by letting them retake their weakened areas.

## ✨ Key Features

### 1. 🎮 Skill Selection Onboarding
Users pick the digital skills they want to be assessed on before starting — no one-size-fits-all tests.

| Skill Category | What It Tests |
|---|---|
| 🔒 Online Safety | Phishing detection, password hygiene, 2FA awareness |
| 📰 Spotting Fake News | Source verification, deepfake awareness, media bias |
| 🛡️ Privacy & Data | Data sharing risks, app permissions, digital footprint |
| 📱 Social Media Literacy | Algorithm awareness, cyberbullying response, screen time |
| 🔍 Search & Research | Boolean search, source evaluation, academic integrity |
| 💻 Coding Basics | HTML/CSS awareness, computational thinking, automation |

### 2. 🤖 AI-Adaptive Assessment Engine
- Generates **8–10 contextual questions** per session using Groq LLaMA
- **60% MCQ + 40% short answer** (tests understanding, not just recall)
- Questions adapt based on selected skills — never generic
- Real-world scenario-based (e.g., "You receive this email — what do you do?")

### 3. 📊 Digital Literacy Scoring & Levels

| Score | Level | Badge | What It Means |
|---|---|---|---|
| 0–40% | Digital Novice | 🌱 | Needs foundational internet safety training |
| 41–65% | Digital Explorer | 🧭 | Aware but has critical gaps in privacy/verification |
| 66–85% | Digital Native | 🛡️ | Strong skills, needs advanced media literacy |
| 86–100% | Digital Champion | 👑 | Ready to teach others and create responsibly |

### 4. 🗺️ Personalized Learning Roadmap
AI generates a **skill-gap analysis** with:
- Specific weak areas identified per skill category
- Curated free resources (courses, videos, tools)
- Estimated time to level up
- Progress milestones

### 5. 💬 Digital Literacy AI Coach
A conversational mentor that:
- Explains *why* an answer was wrong
- Gives real-world examples of digital risks
- Suggests daily habits for better digital hygiene
- Available 24/7 for follow-up questions

### 6. 📄 Digital Literacy Report Card (Export)
- Export results as `.txt` or `.pdf`
- Shareable with teachers, parents, or schools
- Includes score breakdown, skill gaps, and recommendations

## 🚀 Quick Start

### Prerequisites
- Node.js & npm (for local server)
- Supabase Account & Project
- Groq API key

### Installation

```bash
# Clone the repository
git clone https://github.com/team-thunderbuddies/yuvata.git
cd yuvata

# Run local development server
npx serve .
```

### Database Setup (Supabase)
Run the provided `setup.sql` file in your Supabase SQL Editor to initialize the `profiles`, `assessments`, and `chat_history` tables along with the required Row Level Security policies.

## 🛡️ Security

- **Row Level Security (RLS)** implemented via Supabase so users can only access their own data.
- Input sanitization against prompt injection.
- No PII exposed — passwords are fully managed by Supabase Auth (never stored in plaintext).
- Built-in rate limiting and session management.
